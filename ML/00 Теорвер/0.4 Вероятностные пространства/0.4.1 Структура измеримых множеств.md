
### Определение

Вероятностное пространство — это тройка $(\Omega, \mathscr F, P)$

$\Omega = \{ \omega \}$ — множество\пространство элементарных исходов

$\mathscr F \subseteq 2^{\Omega}$ — σ-алгебра на $\Omega$
- $\Omega$ является единицей σ-алгебры $\mathscr F$

$P: \mathscr F \rightarrow [0;1]$ — вероятностная мера
- $P$ σ-аддитивна
- $P(\Omega)=1$


### Теорема о структуре множеств, измеримых по Лебегу

$\mathscr S$ — полукольцо
$\mathscr R(\mathscr S)$ — минимальное полукольцо, содержащее $\mathscr S$

$\mu$ , $\mathscr M$ — мера Лебега и σ-алгебра измеримых множеств, которые были получены в результате продления меры $m$ с полукольца $\mathscr S$.

$\mathcal A \in \mathscr M$
$\mu(\mathcal A) < \infty$

Тогда множество $\mathcal A$ можно представить в виде

$$\mathcal A = \ \bigcap_{i=1}^{\infty}  \bigcup_{j=1}^{\infty} \mathcal A_{ij} \ \backslash \ \mathcal A_0$$
$\mathcal A_{ij} \in \mathscr R(\mathscr S)$
$\mathcal A_0 \in \mathscr M \ , \ \mu(\mathcal A_0) = 0$

Более того, для $\forall i,j$
- $\mathcal A_{i1} \ \subset \ \mathcal A_{i2} \ \subset \ \mathcal A_{i3}, \ ...$ — возрастающая последовательность множеств 
- $\bigcup_{j=1}^{\infty} \mathcal A_{1j} \ \supset \ \bigcup_{j=1}^{\infty} \mathcal A_{2j} \  \supset \ \bigcup_{j=1}^{\infty} \mathcal A_{3j} \ , ...$ — убывающая последовательность множеств 


Доказательство


$\mathcal A \in \mathscr M$ — множество $\mathcal A$ измеримо по условию теоремы

$\mu(\mathcal A) = \mu^*(\mathcal A) = \inf_{ \{ \mathcal D_i \}_{i=1}^{\infty}} \sum_{i=1}^{\infty} \mu(\mathcal D_i) \ , \ \mathcal A \subseteq \bigcup_{i=1}^{\infty} \mathcal D_i$ — инфимум сумм мер по счётным покрытиям множества $\mathcal A$ элементами полукольца $\mathscr S$

По определению инфимума $\forall i \in \Bbb N \ \ \exists \mathcal D_{ij} \ , \ \mathcal D_{ij} \in \mathscr S$ — существует счётное покрытие множества $\mathcal A$ элементами полукольца $\mathscr S$ :
- $\mathcal A \subseteq \bigcup_{j=1}^{\infty} \mathcal D_{ij}$
- $\mu^*(\bigcup_{j=1}^{\infty} \mathcal D_{ij} \ \triangle \ \mathcal A) = \mu^*(\bigcup_{j=1}^{\infty} \mathcal D_{ij} \ \backslash \ \mathcal A) < \frac{1}{i}$

Рассмотрим счётные объединения этих покрытий

$\mathcal C_1 := \bigcup_{j=1}^{\infty} \mathcal D_{1j}$   ←   $\mu^*(\mathcal C_1 \ \backslash \ \mathcal A) < 1$
$\mathcal C_2 := \bigcup_{j=1}^{\infty} \mathcal D_{2j}$   ←   $\mu^*(\mathcal C_2 \ \backslash \ \mathcal A) < \frac{1}{2}$
$\vdots$
$\mathcal C_i := \bigcup_{j=1}^{\infty} \mathcal D_{ij}$   ←   $\mu^*(\mathcal C_i \ \backslash \ \mathcal A) < \frac{1}{i}$
$\vdots$

И рассмотрим конечные пересечения этих счётных объединений

$\mathcal B_1 := \mathcal C_1$
$\mathcal B_2 := \mathcal C_1 \cap \mathcal C_2$
$\vdots$
$\mathcal B_i := \mathcal C_1 \cap \mathcal C_2 \ \cap ... \cap  \ \mathcal C_i$
$\vdots$


С одной стороны, очевидно, что $\mathcal B_i$ — убывающая последовательность.

С другой стороны, мы можем воспользоваться дистрибутивностью операций пересечения и объединения множеств:


$$(\mathcal A_1 \cup \mathcal A_2) \cap (\mathcal B_1 \cup \mathcal B_2) = (\mathcal A_1 \cap \mathcal B_1) \cup (\mathcal A_1 \cap \mathcal B_2) \cup (\mathcal A_2 \cap \mathcal B_1) \cup (\mathcal A_2 \cap \mathcal B_2)$$

А именно:

$\mathcal B_1 = \mathcal C_1 = \bigcup_{j = 1}^{\infty} \mathcal D_{1,j}$


$\mathcal B_2 = \mathcal C_1 \cap \mathcal C_2 =$

	$= \bigcup_{j=1}^{\infty} \mathcal D_{1j} \ \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{2j} =$

	$= \ (\mathcal D_{1,1} \ \cup \ \mathcal D_{1,2} \ \cup \  \mathcal D_{1,3} \ \cup \ ...) \ \cap$
	$\ \cap (\mathcal D_{2,1} \ \cup \ \mathcal D_{2,2} \ \cup \  \mathcal D_{2,3} \ \cup \ ...)=$

	$=\bigcup_{j_1,\ j_2 = 1}^{\infty}(\mathcal D_{1,j_1} \cap \mathcal D_{2,j_2})$


$\mathcal B_3 = \mathcal C_1 \cap \mathcal C_2 \cap \mathcal C_3=$

	$= \bigcup_{j=1}^{\infty} \mathcal D_{1j} \ \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{2j}  \ \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{3j} =$

	$= \ (\mathcal D_{1,1} \ \cup \ \mathcal D_{1,2} \ \cup \  \mathcal D_{1,3} \ \cup \ ...) \ \cap$
	$\ \cap (\mathcal D_{2,1} \ \cup \ \mathcal D_{2,2} \ \cup \  \mathcal D_{2,3} \ \cup \ ...) \ \cap$
	$\ \cap (\mathcal D_{3,1} \ \cup \ \mathcal D_{3,2} \ \cup \  \mathcal D_{3,3} \ \cup \ ...)=$

	$=\bigcup_{j_1, \ j_2, \ j_3 = 1}^{\infty}(\mathcal D_{1,j_1} \cap \mathcal D_{2,j_2} \cap \mathcal D_{3,j_3})$

$\vdots$

$\mathcal B_i = \mathcal C_1 \cap \mathcal C_2 \cap \mathcal C_3  \cap \ ... \ \cap \mathcal C_i=$

	$= \bigcup_{j=1}^{\infty} \mathcal D_{1j} \ \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{2j} \ \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{3j} \ \cap ... \cap \ \bigcup_{j=1}^{\infty} \mathcal D_{ij} =$

	$= \ (\mathcal D_{1,1} \ \cup \ \mathcal D_{1,2} \ \cup \  \mathcal D_{1,3} \ \cup \ ...) \ \cap$
	$\ \cap (\mathcal D_{2,1} \ \cup \ \mathcal D_{2,2} \ \cup \  \mathcal D_{2,3} \ \cup \ ...) \ \cap$
	$\ \cap (\mathcal D_{3,1} \ \cup \ \mathcal D_{3,2} \ \cup \  \mathcal D_{3,3} \ \cup \ ...) \ \cap$
	$\ \ \ \ \ \ \ ...$
	$\ \cap (\mathcal D_{i,1} \ \cup \ \mathcal D_{i,2} \ \cup \  \mathcal D_{i,3} \ \cup \ ...)=$

	$=\bigcup_{j_1, \ j_2, \ j_3, \ ... \ j_i = 1}^{\infty}(\mathcal D_{1,j_1} \cap \mathcal D_{2,j_2} \cap \mathcal D_{3,j_3} \ \cap ... \cap \ \mathcal D_{i,j_i} )$

$\vdots$


Рассмотрим формулу $\mathcal B_i = \bigcup_{j_1, \ j_2, \ j_3, \ ... \ j_i = 1}^{\infty}(\mathcal D_{1,j_1} \cap \mathcal D_{2,j_2} \cap \mathcal D_{3,j_3} \cap ... \cap \mathcal D_{i,j_i} )$ и заметим две вещи.

Во-первых, $\mathcal D_{1,j_1} \cap \mathcal D_{2,j_2} \cap \mathcal D_{3,j_3} \cap ... \cap \mathcal D_{i,j_i} \in \mathscr S$ — поскольку все $\mathcal D_{ij} \in \mathscr S$, а полукольцо замкнуто относительно операции конечного пересечения.

Во-вторых, множество мультииндексов $\{j_1 \ , j_2 \ ,..., \ j_i \}$ не более чем счётно.

Таким образом, множества $\mathcal B_i$ оказывается представимо в виде счётного объединения каких-то элементов полукольца.

$$\mathcal B_i = \bigcup_{k=1}^{\infty} \mathcal E_{ik} \ , \ \mathcal E_{ik} \in \mathscr S$$

И в качестве $\mathcal A_{ij}$ мы можем взять

$$\mathcal A_{ij} := \bigcup_{k=1}^{j} \mathcal E_{ik}$$

Заметим, $\mathcal A_{ij}$ возрастают при любом фиксированном значении $i$.

Заметим, что $\mathcal A_{ij} \in \mathscr R(\mathscr S)$, поскольку кольцо $\mathscr R(\mathscr S)$ состоит из конечных дизъюнктных объединений элементов полукольца $\mathscr S$, и следовательно, любое конечное — не обязательно дизъюнктное —  объединение элементов полукольца $\mathscr S$ всегда будет лежать в кольце $\mathscr R(\mathscr S)$.

Устремим $i \to \infty$ и введём ещё одно обозначение:

$$\mathcal B_i \rightarrow \mathcal B_{\infty} := \bigcap_{l=1}^{\infty} \mathcal C_l = \bigcap_{l=1}^{\infty} \bigcup_{k=1}^{\infty} \mathcal A_{lk}$$

Заметим, что $\mathcal B_{\infty} \in \mathscr M$ по построению.

Заметим, что

$$\mu(\mathcal B_{\infty}  \backslash \ \mathcal A) \le \mu(\mathcal B_{i} \  \backslash \ \mathcal A) < \frac{1}{i} \ , \ \forall i \Rightarrow \mu(\mathcal B_{\infty}  \backslash \ \mathcal A) = 0$$

А следовательно, в качестве $\mathcal A_{0}$ мы можем взять

$$\mathcal A_0 = \mathcal B_{\infty} \backslash \ \mathcal A$$
