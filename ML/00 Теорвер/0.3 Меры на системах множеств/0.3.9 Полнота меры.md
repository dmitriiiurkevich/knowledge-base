
### Определение

Меру $\mu$, определённую на σ-алгебре $\mathscr M$ будем называть полной, если

$$\mu(\mathcal A) = 0 \ , \ \mathcal B \subseteq \mathcal A \Rightarrow \left\{ \begin{array}{l} \mathcal B \in \mathscr M \\ \mu(\mathcal B) = 0 \end{array} \right.$$

### Замечание

Любая мера Лебега полна, поскольку

$$\mu(\mathcal A) = 0 \Rightarrow \forall \varepsilon > 0 \ \ \forall \mathcal B \subseteq \mathcal A \ \ \exists \ \emptyset \ : \ \mu^*(\mathcal B \ \triangle \ \emptyset) = \mu^*(\mathcal B) \le \mu^*(\mathcal A) = 0 < \varepsilon $$
