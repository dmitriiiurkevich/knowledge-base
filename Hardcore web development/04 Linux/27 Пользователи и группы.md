`$ id`
`$ id my_user`

`$ groups`
`$ groups my_user`
`$ groups $whoami`

С каждым пользователем связано два идентификатора:
- *uid* — идентификатор пользователя
- *gid* — идентификатор дефолтной группы пользователя — *default group* или *primary group*

Каждый пользователь может состоять в большом количестве разнообразных групп. Тем не менее, дефолтная группа у каждого пользователя в каждый момент времени может быть только одна.  

```
$ groups dmitrii
dmitrii : dmitrii adm dialout cdrom floppy sudo audio dip video plugdev netdev docker
```

```
$ id dmitrii
uid=1000(dmitrii) gid=1000(dmitrii) groups=1000(dmitrii),4(adm),20(dialout),24(cdrom),25(floppy),27(sudo),29(audio),30(dip),44(video),46(plugdev),116(netdev),999(docker)
```



При создании пользователя автоматически создаётся одноимённая с ним группа, единственным членом которой будет сам пользователь. Эта группа будет являться дефолтной группой только что созданного пользователя. В дальнейшем значение дефолтной группы пользователя может быть изменено с помощью одной из специальных команд. 

`/etc/passwd` — файл, содержащий разнообразную справочную информацию о каждом пользователе системы, включая его *uid* и *gid*.

`/etc/group` — файл, содержащий содержащий для каждого пользователя системы информацию обо всех группах, в которые он входит.

## Создание и удаление пользователей

`$ sudo adduser my_user`

`$ sudo echo 'my_user ALL=(ALL:ALL) ALL' >> etc/sudoers` — добавит записи о новом пользователе в файл *sudoers*

## Создание и удаление групп

`$ sudo groupadd my_group`

Добавление нового пользователя в группу *my_group*

`$ sudo adduser my_user my_group`
`$ sudo usermod -aG my_group my_user`

`$ sudo usermod -g my_group my_user`
## Изменение паролей пользователей

`$ passwd`
`$ sudo passwd my_user`

`/etc/shadow` — файл, в котором в зашифрованном виде хранятся пароли пользователей.
