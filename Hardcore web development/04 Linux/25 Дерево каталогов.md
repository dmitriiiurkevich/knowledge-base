
*The Filesystem Hierarchy Standard (FHS)*
*Linux Foundation*, 1991


![[d7b6719b-3e9c-4441-9024-6692b8b57e1a.png]]


*Root*
00.  `/`

*Essential programs*
01. `/bin` — user binaries  — essential binaries that all users have access to
	 sym-linked to `/usr/bin`
	 содержит исполняемые файлы основных утилит командной строки

02. `/sbin` — root binaries — essential binaries that available to root user
	 sym-linked to `/usr/sbin`
	 содержит вспомогательные утилиты для администрирования и настройки системы: *iptables*, *useradd*,...
	 обычно не включается в *PATH*

03. `/lib` — essential libraries for binaries in `/bin` and `/sbin` 
	 sym-linked to `/usr/lib`

*Non-Essential programs*
*System specific*
04. `/opt` — additional software — programs not found in distribution repositories
	 сюда чаще всего ставится дополнительное ПО, чаще всего, проприетарное 

05. `/etc` — configuration files for programs in `/opt` and `/usr` 
	содержит все общесистемные конфигурационные файлы
	если пользователь сохранит отредактированный конфигурационный файл в своей домашней директории, то такая копия будет иметь для системы больший приоритет

*Portable*
06. `/usr` — portable non-essential programs and program files
	фактически

*User Directories*
07. `/home` — normal users home directory

08. `/root` — root user home directory

*Kernel File Systems*
directories populated by the kernel to provide information to programs and users
09. `/proc` — information about processes

10. `/sys` — information about system hardware and kernel

11. `/run` — information about the system since the last boot

*Mount Points*
12. `/media` — removable media — CD-drives

13. `/mnt` — temporary file systems — USB-drives

14. `/tmp` — pseudo-filesystem for temporary files

*Other Directories*
15. `/boot` — files essential for booting the system — *initrd*, kernel, bootloader configuration

16. `/dev` — files used for devices — HDD, streams: *stdin*, *stdout*, *stderr*

17. `/srv` — files used for services — www, ftp, rsync

18. `/var` — variable (changing) files — logs, lock files
