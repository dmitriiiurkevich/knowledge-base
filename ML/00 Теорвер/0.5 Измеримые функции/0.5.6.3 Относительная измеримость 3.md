
### Примеры

#### Пример 1

$\mathcal X = \{1, 2, 3, 4, 5, 6\}$ — исходы броска игральной кости

$f: \mathcal X \to \{0,1,2,3\}$
$f(x)=x(mod \ 4)$

$g: \mathcal X \to \{0,1\}$
$g(x)=x(mod \ 2)$

$\Large \sigma \normalsize (f) = \{  \emptyset,\ \{1,5\},\ \{2,6\},\ \{3\}, \ \{4\}, \ \{1,2,5,6\}, \ \{1,3,5\}, \ \{1,4,5\}, \ \{2,3,6\}, \ \{2,4,6\}, \ \{3,4\}, \ \{1,2, 3, 5,6\}, \ \{1,2,4, 5, 6\}, \ \{2,3,4, 6\}, \ \{1,5,3, 4\}, \ \mathcal X \}$
$\Large \sigma \normalsize (g) = \{  \emptyset,\ \{1,3,5\},\ \{2,4,6\}, \ \mathcal X \}$


$\Large \sigma \normalsize (g) \subseteq \Large \sigma \normalsize (f) \Rightarrow$ отображение $g$ является $\Large \sigma \normalsize (f)$-измеримым. 


$h: \{0,1,2,3\} \to \{0,1\}$
$h(x)=x(mod \ 2)$
$g = h \circ f \Rightarrow$ отображение $g$ является $\Large \sigma \normalsize (f)$-измеримым.


Функция $f(x)=x(mod \ 4)$ породила более богатую алгебру $\Large \sigma \normalsize (f)$ — 16 подмножеств $\mathcal X$.

Функция $g(x)=x(mod \ 2)$ породила более бедную алгебру $\Large \sigma \normalsize (g)$ — 4 подмножества $\mathcal X$.

Обе алгебры являются подалгебрами полной алгебры $2^{\mathcal X}$ — 64 подмножества $\mathcal X$.

Алгебра $\Large \sigma \normalsize (g)$ является подалгеброй $\Large \sigma \normalsize (f)$.


#### Пример 2

$\mathcal X = \{1, 2, 3, 4, 5, 6\}$ — исходы броска игральной кости

$f: \mathcal X \to \{0,1,2\}$
$f(x)=x(mod \ 3)$

$g: \mathcal X \to \{0,1\}$
$g(x)=x(mod \ 2)$

$\Large \sigma \normalsize (f) = \{  \emptyset,\ \{1,4\},\ \{2,5\},\ \{3, 6\}, \ \{1,2,4,5\}, \ \{1,3,4,6\}, \ \{2,3,5,6\}, \ \mathcal X \}$

$\Large \sigma \normalsize (g) = \{  \emptyset,\ \{1,3,5\},\ \{2,4,6\}, \ \mathcal X \}$

Всё так же, как и в первом примере, но алгебра $\Large \sigma \normalsize (g)$ больше не является подалгеброй $\Large \sigma \normalsize (f)$.



*N.B.*
Информация, которая говорит нам, что 
- по остатку по модулю 4 можно однозначно восстановить остаток по модулю 2
- по остатку по модулю 3 нельзя однозначно восстановить остаток по модулю 2
закодирована в структурах σ-алгебр, порождаемых соответствующими случайными величинами.

σ-Алгебра — это формализация понятия «информация» или «знание». Операции над σ-алгебрами соответствуют операциям над информацией: объединение — «знать и то, и другое», пересечение — «знать только общее».

Это мощный язык, который позволяет говорить об информации чисто математически, не обращаясь к вульгарной интуиции.


#### Пример 3

$f: \big(\Bbb R, \mathfrak B(\Bbb R)\big) \rightarrow \big(\Bbb R, \mathfrak B(\Bbb R)\big) \quad f(x)=x^2$
$g: \big(\Bbb R, \mathfrak B(\Bbb R)\big) \rightarrow \big(\Bbb R, \mathfrak B(\Bbb R)\big) \quad g(x)=|x|$

Рассмотрим $\mathcal B \in \mathfrak B(\Bbb R)$

$\Large \sigma \normalsize (f) = \{f^{-1}(\mathcal B) : \mathcal B \in \mathfrak B(\Bbb R)\} = \{x: x^2 \in \mathcal B\}$
$\Large \sigma \normalsize (g) = \{g^{-1}(\mathcal B) : \mathcal B \in \mathfrak B(\Bbb R)\} = \{x: |x| \in \mathcal B\}$

$\Large \sigma \normalsize (f) = \Large \sigma \normalsize (g)$ — σ-алгебра, порождённая симметричными относительно нуля борелевскими множествами

$$\mathcal A \in \mathfrak B(\Bbb R)$$
$$\Downarrow$$
$$\mathcal A \in \Large \sigma \normalsize (f) \iff \mathcal A = -\mathcal A$$
$$\mathcal A \in \Large \sigma \normalsize (g) \iff \mathcal A = -\mathcal A$$

Вывод: измеримые функции $f$ и $g$ несут в себе одну и ту же информацию — по значению функции $f$ можно однозначно восстановить значение функции $g$, и наоборот.

$$g = h \circ f \quad h = \sqrt{x}$$


#### Пример 4

$f: \big(\Bbb R, \mathfrak B(\Bbb R)\big) \rightarrow \big(\Bbb R, \mathfrak B(\Bbb R)\big) \quad f(x)= \sin(x)$
$g: \big(\Bbb R, \mathfrak B(\Bbb R)\big) \rightarrow \big(\Bbb R, \mathfrak B(\Bbb R)\big) \quad g(x)= \sin(2x)$

Атомарные элементы σ-алгебры $\Large \sigma \normalsize (f)$ — это классы эквивалентности отношения

$$x_1 \sim  x_2 \iff \sin(x_1)=\sin(x_2)$$

они называются слоями отображения — *fibers* —   более точно: для любого значения $c \in [-1;1]$ слой определяется как множество

$$f^{-1}(c) = \{x : \sin(x)=c\}$$
$$g^{-1}(c) = \{x : \sin(2x)=c\}$$

Вся вещественная прямая $\Bbb R$ разбивается в произведение таких слоёв.

$\Large \sigma \normalsize (f)$ — σ-алгебра, порождённая всеми слоями отображения $f$.
$\Large \sigma \normalsize (g)$ — σ-алгебра, порождённая всеми слоями отображения $g$.

То есть, множество $\mathcal A \in \Large \sigma \normalsize (f)$ если и только если множество $\mathcal A$ является объединением какого-то количества слоёв отображения $f$. 

Что полностью соответствует формальному определению:

$$\mathcal A \in \Large \sigma \normalsize (f) \iff \exists \ \mathcal B \in \mathfrak B\big([-1;1]\big) : \mathcal A = f^{-1}(\mathcal B)$$

И аналогично для $\Large \sigma \normalsize (g)$.


Для того, чтобы убедиться в том, что σ-алгебры $\Large \sigma \normalsize (f)$ и $\Large \sigma \normalsize (g)$ не вкладываются друг в друга, достаточно просто взглянуть на слои каждого из этих отображений.


$$f^{-1}(c) = \{\arcsin(c) + 2\pi \Bbb Z\} \cup \{\pi-\arcsin(c) + 2\pi \Bbb Z\}$$
$$f^{-1}(1)=\{\frac{\pi}{2} + 2\pi \Bbb Z\}$$
$$f^{-1}(-1)=\{-\frac{\pi}{2} + 2\pi \Bbb Z\}$$

$$g^{-1}(c) = \{\frac{\arcsin(c)}{2} + \pi \Bbb Z\} \cup \{\frac{\pi-\arcsin(c)}{2} + \pi \Bbb Z\}$$
$$g^{-1}(1)=\{\frac{\pi}{4} + \pi \Bbb Z\}$$
$$g^{-1}(-1)=\{-\frac{\pi}{4} + \pi \Bbb Z\}$$
