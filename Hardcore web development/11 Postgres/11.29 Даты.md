`DATE` — год, месяц, день
`TIME` — часы, минуты, секунды, доли секунды
`TIME with time zone` — часы, минуты, секунды, доли секунды + time zone
`TIMESTAMP` — год, месяц, день, часы, минуты, секунды, доли секунды
`TIMESTAMPTZ` — год, месяц, день, часы, минуты, секунды, доли секунды + time zone
`INTERVAL` — для хранения промежутка времени.

Timestamptz

``` sql
select my_timestamptz at time zone 'Europe/Sofia' from my_table;
select my_timestamptz at time zone 'UTC' from my_table;
```

Interval

```sql
select pg_typeof('2024-02-01'::date - '2023-01-01'::date); --> integer

select pg_typeof('2024-02-01'::timestamp - '2023-01-01'::timestamp); --> interval
```

Так можно, но ненужно

```sql
select now(), now() + '1 hour';
select now(), now() + '1 day';
select now(), now() + '1 month';
```

Так нельзя

```sql
select now(), now() - '1 month'; -- просит привести к типу interval явно
```

Так нужно

```sql
select now(), now() + '1 month'::interval;
select now(), now() + '1 hour'::interval;
select now(), now() + '1 year'::interval;

select now(), now() - '1 month'::interval;
select now(), now() - '1 hour'::interval;
select now(), now() - '1 year'::interval;
```

```sql
select now(), now() - '1 year 1 month'::interval;
select now(), now() - '1 year 1 month 1 hour'::interval;
select now(), now() - '1 year 1 month 1 hour 1 minute'::interval;
select now(), now() - '1 year 1 month 1 hour 1 minute 1 second'::interval;
```

Арифметические действия с интервалами

```sql
select now() - (now() + '1 hour'); --> interval: -01:00:00

select '2024-06-24 03:38:00'::timestamp - '2024-06-24 01:38:00'::timestamp;
select '2024-06-24 03:38:00'::timestamp - '2023-05-01 14:13:28'::timestamp;

select '2024-06-24 03:38:00'::timestamp - '2023-05-01 14:13:28'::timestamp 
> '1 day'::interval; --> true

select '2024-06-24 03:38:00'::timestamp - '2023-05-01 14:13:28'::timestamp 
> '1000 day'::interval; --> false
```

Функции `age` и `extract`

``` sql
select age('2024-06-21 01:38:11'::timestamp, '2021-01-24 01:12:48'::timestamp);
--> 3 years 4 mons 28 days 00:25:23

select extract(day from 
age('2024-06-21 01:38:11'::timestamp, '2021-01-24 01:12:48'::timestamp));
--> 28

select extract(days from 
age('2024-06-21 01:38:11'::timestamp, '2021-01-24 01:12:48'::timestamp));
--> 28

select extract(years 
from age('2024-06-21 01:38:11'::timestamp, '2021-01-24 01:12:48'::timestamp));
--> 3
```

``` sql
select
extract (year from now()),
extract (month from now()),
extract (day from now()),
extract (hour from now()),
extract (minute from now()),
extract (second from now());
```

*N.B.*: Функция `now()` не описана в стандарте *SQL*. Поэтому, если хотите писать более переносимый код, то есть код, который будет работать и в других СУБД, то вместо `now()`лучше использовать `CURRENT_TIMESTAMP`.

Функция `date_trunk`

``` sql
select now(), date_trunc('month', now());
--> 2024-06-25 04:09:33.672 +0300    2024-06-01 00:00:00.000 +0300

select now(), date_trunc('year', now());
--> 2024-06-25 04:09:46.959 +0300    2024-01-01 00:00:00.000 +0300

select now(), date_trunc('century', now());
--> 2024-06-25 04:09:46.959 +0300    2001-01-01 00:00:00.000 +0300
```

Как получить последний день месяца?

``` sql
select date_trunc('month', now()) + interval '1 month - 1 day' as last_day;
```
