
Python program
$\quad \downarrow$
Physical lines of code — ending with a physical new-line character
$\quad \downarrow$
Logical lines of code — tokenized — ending with a  logical NEWLINE token


Неявная конвенция преобразования физических строк в логические применяется для
- list literals
- tuple literals
- dictionary literals
- set literals
- function arguments/parameters


``` python
lst = [1,  # some comment
       2,
       3]
```


``` python
def my_function(a,  # some comment
                b,  
                c):
   ...

my_function(10,   # some comment

            20,

            30)
```


Во всех остальных случаях применяется явная конвенция — многострочные операторы не преобразуются неявно в одну логическую строку.

### Multiline statement

``` python
# оператор можно явным образом разбить на несколько физических строк
# используя символ backslash

def my_function(a, b):
   return \
       a+b
```


``` python
# не важна величина отступа во всех физических строках
# относящихся к одной и той же логической строке
# кроме самой первой

if a \
   and b \
   and c:
    pass
```


Комментарии не могут быть частью multiline statement, впрочем, так же как и частью singleline statement.

``` python
if a \
   and b \  #error: no comments here
   and c:
    pass
```
