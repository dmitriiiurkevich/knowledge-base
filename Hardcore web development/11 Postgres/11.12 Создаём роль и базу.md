Пользователь — это роль, имеющая право логина. Не каждая роль имеет право логина. Роль, имеющая право логина, называется *пользователем*.

```sql
# CREATE DATABASE rroom_db
    WITH
    TEMPLATE=template0
    ENCODING='UTF8'
    LC_COLLATE='ru_RU.UTF-8'
    LC_CTYPE='ru_RU.UTF-8'
    owner rroom;
CREATE DATABASE
```

- `LC_COLLATE` — определяет порядок следования символов алфавитов при сортировке
- `LC_CTYPE` — определяет, какие символы рассматриваются как часть алфавита, как цифры и т. д. — для правильной работы с верхним и нижним регистром
- `LC_MONETARY` — определяет денежные единицы и их форматирование денежных числовых значений
- `LC_NUMERIC` — определяет форматирование неденежных числовых значений, например, разделитель разрядов
- `LC_TIME` — определяет формат даты и времени
- `LC_MESSAGES` — определяет язык программных сообщений
