
### Python Type Hierarchy

- Numbers
	- Integral Numbers
		- integers
		- booleans
	- Non-Integral Numbers
		- floats (IEEE 754 double precision binary float)   ← вещественные числа, аналог double в Java and C++
		- complex
		- decimals   ← вещественные числа в виде честных десятичных дробей
		- fractions   ← рациональные числа в виде простых дробей
- Mutable Collections  
	- Sequences
		- lists
	- Sets 
		- sets
	- Mappings
		- dictionaries
- Immutable Collections
	- Sequences
		- tuples
		- strings
	- Sets
		- frozensets
	- Mappings
		- пока нет, но, возможно, скоро появятся
- Callables
	- user-defined functions
	- generators
	- classes
	- instance methods
	- class instances  ← инстансы классов, снабженные функцией `__call__`
	- built-in functions  ← такие как `len()`, `open()`, …
	- built-in methods ← такие как `my_list.append()`, …
- Singletons
	- `None`
	- `NotImplemented`
	- `Ellipsis`
