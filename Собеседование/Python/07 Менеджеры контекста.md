
### Протокол контекстного менеджера

Любой объект может стать контекстным менеджером, если он реализует два специальных метода:
- `__enter__()` — вызывается при входе в блок `with`
- `__exit__(exc_type, exc_value, traceback)` — вызывается при выходе из блока `with`

### Как это работает пошагово?

```python
with context_manager as variable:
    # код внутри блока
    pass
```

1. Вызывается `context_manager.__enter__()`
2. Результат `__enter__()` присваивается переменной `variable` (если указана)
3. Выполняется код внутри блока `with`
4. При выходе из блока (нормальном или через исключение) вызывается `__exit__()`
