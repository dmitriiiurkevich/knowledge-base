Систему множеств $\mathscr S$ будем называть *полукольцом* — *semiring* —если она обладает следующими тремя свойствами:
1. $\emptyset \in \mathscr S$
2. $\mathcal A, \mathcal B \in \mathscr S \Rightarrow \mathcal A \cap \mathcal B \in \mathscr S$
3. $\mathcal A_1, \mathcal A \in \mathscr S, \mathcal A_1 \subset \mathcal A \Rightarrow \exists \mathcal A_2 ... \mathcal A_n \in \mathscr S: \mathcal A = \bigsqcup_{i=1}^{n} \mathcal A_i$

*N.B.*
В третьем свойстве существенным является то, что множества $\mathcal A_1 ... \mathcal A_n$ дизъюнктны. Впоследствии мы захотим определить аддитивную меру μ на полукольце так, чтобы выполнялось соотношение μ(A) = μ(A₁) + μ(A₂) + ... + μ(Aₙ)

### Примеры полуколец

1.
Полуоткрытые интервалы в $\Bbb R$
Полуоткрытые интервалы $\subseteq [a,b)$
1. $\emptyset =  [a; a)$
2. Пересечение двух полуоткрытых интервала тоже является полуоткрытым интервалом, возможно, — пустым.
3. $[a_1; b_1) \subset [a;b) \Rightarrow [a;b)=[a;a_1) \sqcup [a_1;b_1) \sqcup [b_1;b)$

2.
Полуоткрытые ячейки в $\Bbb R^n$
$[a_1; b_1) \times [a_2; b_2) \times ... \times [a_n; b_n)$

3.
Интервалы всех типов в в $\Bbb R$

4.
Выпуклые многоугольники/многогранники

5.
Конечные подмножества не более чем счётного множества

6.
Пусть
$f: \mathcal X \rightarrow \Bbb R$ — измеримая функция
$\Bbb I$ — интервалы в $\Bbb R$ — см. пункты 1 и 3

Тогда, полукольцо образуют множества вида $f^{-1}(I), \ I \in \Bbb I$.

6+1.
Пусть
$f: \mathcal X \rightarrow \mathcal Y$ — измеримая функция
$\mathscr E$ — полукольцо в $\mathcal Y$ 

Тогда, полукольцо образуют множества вида $f^{-1}(E), \ E \in \mathscr E$ — т. н. полукольцо полных прообразов элементов полукольца под действием измеримой функции.


*N.B.*
Пересечение двух полуколец может не быть полукольцом:

$X = \{1, 2, 3, 4\}$
$\mathscr S_1 = \{\emptyset, \{1\}, \{2\}, \{3,4\}, \{1,2,3,4\}\}$ 
$\mathscr S_2 = \{\emptyset, \{1\},\{2,3\}, \{4\},\{1,2,3,4\}\}$ 

$\mathscr S_1 \cap \mathscr S_2 = \{\emptyset, \{1\}, \{1,2,3,4\}\}$

В $\mathscr S_1 \cap \mathscr S_2$ не выполнено третье свойство: $\{1\}$ не получится дизъюнктно расширить до $\{1,2,3,4\}$.

### Фундаментальные свойства полуколец

#### Свойство 1

$\mathscr S$ — полукольцо
$\mathcal A, \mathcal B \in \mathscr S$ — два произвольных элемента полукольца

$\mathcal A = (\mathcal A \cap \mathcal B) \sqcup (\mathcal A \backslash \mathcal B)$

$\mathcal A \cap \mathcal B$ — также является элементом полукольца — по второму свойству полуколец

$\mathcal A \ \backslash \ \mathcal B$ — не обязательно является элементом полукольца, однако, представимо в виде конечного дизъюнктного объединение каких-то элементов полукольца — по третьему свойству полуколец, поскольку $\mathcal A \cap \mathcal B \subset \mathcal A$

Таким образом, любая пара элементов полукольца — $\mathcal A, \mathcal B$ — распадается на
1. элемент полукольца $= \mathcal A \cap \mathcal B$
2. конечное дизъюнктное объединение элементов полукольца $=\mathcal A \backslash \mathcal B$
3. конечное дизъюнктное объединение элементов полукольца $=\mathcal B \backslash \mathcal A$

#### Свойство 2

$\mathcal A = \bigsqcup_i \mathcal A_i$
$\mathcal B = \bigsqcup_j \mathcal B_j$

$\Rightarrow \mathcal A \cap \mathcal B = \bigsqcup_{i,j} \mathcal A_i \cap \mathcal B_j$

$\Rightarrow \mathcal A \ \backslash \ \mathcal B = \bigsqcup_i (\mathcal A_i \ \backslash  \ \mathcal B) = \bigsqcup_i (\mathcal A_i \ \backslash \ \bigsqcup_j \mathcal B_j)$

$\Rightarrow \mathcal B \ \backslash \ \mathcal A = \bigsqcup_j (\mathcal B_j \ \backslash  \ \mathcal A) = \bigsqcup_j (\mathcal B_j \ \backslash \ \bigsqcup_i \mathcal A_i)$

$\Rightarrow \mathcal A \ \triangle \ \mathcal B = (\mathcal A \ \backslash \ \mathcal B) \ \sqcup \ (\mathcal B \ \backslash \ \mathcal A) =$
$= (\bigsqcup_i (\mathcal A_i \ \backslash \ \bigsqcup_j \mathcal B_j)) \ \sqcup \ (\bigsqcup_j (\mathcal B_j \ \backslash \ \bigsqcup_i \mathcal A_i)) =$
$= (\bigsqcup_i (\mathcal A_i \ \backslash \ \bigsqcup_j \mathcal B_j)) \ \sqcup \ (\bigsqcup_j (\mathcal B_j \ \backslash \ \bigsqcup_i \mathcal A_i))$

$\Rightarrow \mathcal A \cup \mathcal B = (\mathcal A \cap \mathcal B) \ \sqcup \ (\mathcal A \ \backslash \ \mathcal B) \ \sqcup \ (\mathcal B \ \backslash \ \mathcal A) =$
$=(\bigsqcup_{i,j} \mathcal A_i \cap \mathcal B_j) \ \sqcup \ (\bigsqcup_i (\mathcal A_i \ \backslash \ \mathcal B)) \ \sqcup \ (\bigsqcup_j (\mathcal B_j \ \backslash \ \mathcal A)) =$
$=(\bigsqcup_{i,j} \mathcal A_i \cap \mathcal B_j) \ \sqcup \ (\bigsqcup_i (\mathcal A_i \ \backslash \ \bigsqcup_j \mathcal B_j)) \ \sqcup \ (\bigsqcup_j (\mathcal B_j \ \backslash \ \bigsqcup_i \mathcal A_i))$


Здесь, как и в предыдущем свойстве, 
- пересечения и дизъюнктные объединения элементов полукольца также являются элементами полукольца
- разности и объединения элементов полукольца, вообще говоря, могут и не являться элементами полукольца, однако представимы в виде конечных дизъюнктных объединений элементов полукольца.

Пересечение двух множеств автоматически дизъюнктно и дистрибутивно. Результаты всех остальных теоретико-множественных операций "атомизируются" — раскладываются на дизъюнктные компоненты.

В результате, любая операция над элементами полукольца сводится к комбинированию "атомарных" элементов исходного полукольца.
