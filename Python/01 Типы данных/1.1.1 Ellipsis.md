2\. Альтернатива `None`

``` python
def nth_odd(n):
    if isinstance(n, int):
        return 2 * n - 1
    else:
	   return None


def original_num(m = ...):
    if m is ...:
         print('This function needs some input')
       
    if m is None:
        print('Non integer input provided to nth_odd() function')
        
    if isinstance(m, int):
        if m % 2:
            print(f'{m} is {int((m + 1) / 2)}th odd number')
        else:
            print(f'{m} is not an odd number')
```


3\. Слайсы массивов в *NumPy*

``` python
import numpy as np

a = np.arange(16).reshape(2, 2, 2, 2)

print(a[..., 0].flatten())
print(a[:, :, :, 0].flatten())

# output:
# [0  2  4  6  8 10 12 14]
# [0  2  4  6  8 10 12 14]
```
