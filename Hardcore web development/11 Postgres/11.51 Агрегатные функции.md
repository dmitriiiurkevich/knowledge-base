
Агрегатные функции — это функции, которые преобразуют столбец значений в одно результирующее значение.
## count

```SQL
select count(*) from my_table; 
select count(1) from my_table; 
--> total amount of lines

select count(my_column) from my_table;
--> amount of lines where my_column != null
```

## sum, avg, min, max


```SQL
select sum(some_int) from (
    select unnest(array[10, 20, 30]) -- тут должен быть содержательный запрос
);                                   -- или таблица
--> 60                               -- unnest(array...) только для примера
```

## array_agg

``` SQL
select array_agg(some_int) from (
    select unnest(array[10, 20, 30]) some_int
);
```

```
 array_agg
------------
 {10,20,30}
```


Более жизненный пример с реальной таблицей.

```SQL
CREATE TABLE IF NOT EXISTS student
(
    student_id bigint GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
    name varchar(150) NOT NULL,
    hobby varchar(150)
);
```

```SQL
INSERT INTO student (name, hobby)
VALUES 
('Smith', 'Swimming'), ('Smith', 'Hiking'), ('Smith', 'Cycling'), 
('Brown', 'Cycling'), ('Brown', 'Running'), 
('Miller', 'Skiing'), ('Miller', null);
```

```SQL
SELECT name, array_agg(hobby) AS hobbyes
FROM student
WHERE hobby IS NOT null
GROUP BY name;
```

```
  name  |          hobbyes
--------+---------------------------
 Smith  | {Swimming,Hiking,Cycling}
 Miller | {Skiing}
 Brown  | {Cycling,Running}
```

```SQL
SELECT hobby, array_agg(name) AS names
FROM student
WHERE hobby IS NOT null
GROUP BY hobby;
```

Элементы массива можно упорядочить

```SQL
SELECT name, array_agg(hobby ORDER BY hobby ASC) as hobbyes
FROM student
WHERE hobby IS NOT null
GROUP BY name;
```

Элементы массива можно отфильтровать

```SQL
SELECT name, array_remove(array_agg(hobby ORDER BY hobby ASC), null) AS hobbyes
FROM student
GROUP BY name;
```

```SQL
SELECT name, array_agg(hobby ORDER BY hobby ASC)
filter (WHERE hobby IS NOT null) as hobbyes
FROM student
GROUP BY name;
```

Также возможно реализовать и более сложные варианты фильтрации с использованием *CASE*. Но об этом позже.

## string_agg

``` SQL
select string_agg(some_int::varchar, ', ') from (
    select unnest(array[10, 20, 30]) some_int
);
-- int необходимо конвертировать в varchar или text
```

```
 string_agg
------------
 10, 20, 30
```



```SQL
SELECT name, string_agg(hobby, ', ') AS hobbyes
FROM student
WHERE hobby IS NOT null
GROUP BY name;
```


## json_agg, jsonb_agg 

```SQL
select json_agg(some_int) from (
    select unnest(array['Alex', 'Peter', 'Ivan']) some_int
);
--> json
```


```
        json_agg
---------------------------
 ["Alex", "Peter", "Ivan"]
```


```SQl
select json_agg(t.*) from (
    select unnest(array['Alex', 'Petr', 'Ivan']) name
) t;
```


```
        json_agg
------------------------
 [{"name":"Alex"}, +
  {"name":"Petr"}, +
  {"name":"Ivan"}]
(1 row)
```
