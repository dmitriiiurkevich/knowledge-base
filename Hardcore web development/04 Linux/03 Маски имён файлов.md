`$ echo ~` — напечатает адрес домашнего каталога

`$ echo *` — напечатает список всех нескрытых файлов в текущей директории 

`$ echo *.txt`
`$ echo 4*`
`$ echo ?.txt`
`$ echo ??.txt`
`$ echo 4??.txt` — напечатает список всех файлов в текущей директории, имена которых соответствуют заданной маске

Перед началом выполнения команды оболочка заменяет маску на соответствующий ей список имён файлов.

`$ touch {1..50}.txt` — создаст 50 файлов с именами *1.txt* ... *50.txt*
`$ touch {john, alex, simon}.txt` — создаст 3 файла с именами *john.txt*, *alex.txt*, *simon.txt*

`$ mkdir -p {2020..2024}/{01..12}` — создаст 5 каталогов с именами *2020* ... *2024*, в каждом из них создаст 12 каталогов с именами *01* ... *12*

Важно различать два способа задания диапазонов значений в масках имён файлов:
- `{1..3}.txt` — диапазон, заданный с помощью фигурных скобок, будет преобразован в список имён *1.txt*, *2.txt*, *3.txt* 
- `[1-3].txt` — диапазон, заданный с помощью квадратных скобок, будет преобразован в  спиок имён *1.txt*, *2.txt*, *3.txt*, пересечённый со списком имён действительно существующих файлов

`$ for i in {0..9}; do mv "$i.txt" "$i.jpg"; done` — переименует десять файлов, при условии, что файлы с такими именами действительно существуют
