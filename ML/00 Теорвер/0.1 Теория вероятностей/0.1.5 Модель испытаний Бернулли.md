Случайный эксперимент — серия из $n$ независимых экспериментов с двумя исходами.

Пространство элементарных исходов:
$$\Omega = \{ \omega = (i_1,...,i_n) \in \{0, 1\}^n\}$$
$$\Vert \Omega \Vert = 2^n$$
Вероятностная мера:
$$p(\omega) = p^{\sum_{k=1}^{n} \omega_k } \cdot (1-p)^{(n-\sum_{k=1}^{n} \omega_k) }$$

Поскольку, $\Vert \Omega \Vert < \infty$, то 
$$p(\mathcal A) = \sum_{\omega \in \mathcal A} p(\omega)$$
