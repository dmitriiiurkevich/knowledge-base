Миф: *Python* является высокоуровневым языком программирования.

Давайте посмотрим, на каких языках написан код библиоетки *numpy*:
- 60.0% — Python
- 35.7% — C
- 2.5% — C++
- 1.0% — Cython
- 0.5% — Meson
- 0.2% — Fortran
- 0.1% — Other

Основа *numpy* написана на *C* и использует *Python C-API* для того, чтобы работать с *Python*.

В *Python* есть две равные по значимости части:
- *Python API* — высокоуровневая часть Python — позволяет писать простой код
- *Python C-API* — низкоуровневая часть Python — позволяет писать быстрый код

С помощью *Python CAPI* пользователь может написать модуль на чистом *C*, используя сторонние библиотеки, скомпилировать его, а затем импортировать его в свой *Python* код.

Более того, даже отдельные операторы Python — например, join — могут быть частично реализованы на *C*.

Реальность: в языке *Python* реализованы апи высокого и низкого уровня.
