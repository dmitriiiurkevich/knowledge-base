
### Python Type Hierarchy

- Numbers
	- Integral Numbers
		- integers
		- booleans
	- Non-Integral Numbers
		- floats (IEEE 754 double precision binary float)   ← вещественные числа, аналог double в Java and C++
		- complex
		- decimals   ← вещественные числа в виде честных десятичных дробей
		- fractions   ← рациональные числа в виде простых дробей
- Mutable Collections  
	- Sequences
		- lists
	- Sets 
		- sets
	- Mappings
		- dictionaries
- Immutable Collections
	- Sequences
		- tuples
		- strings
	- Sets
		- frozensets
	- Mappings
		- пока нет, но, возможно, скоро появятся
- Callables
	- user-defined functions
	- generators
	- classes
	- instance methods
	- class instances  ← инстансы классов, снабженные функцией `__call__`
	- built-in functions  ← такие как `len()`, `open()`, …
	- built-in methods ← такие как `my_list.append()`, …
- Singletons
	- `None`
	- `NotImplemented`
	- `Ellipsis`



### Три примера применения`Ellipsis`. 


1\. Альтернатива `pass`

``` python
def some_func():
    ...

def another_func():
    pass
```


2\. Альтернатива `None`

``` python
def nth_odd(n):
    if isinstance(n, int):
        return 2 * n - 1
    else:
	   return None


def original_num(m = ...):
    if m is ...:
         print('This function needs some input')
       
    if m is None:
        print('Non integer input provided to nth_odd() function')
        
    if isinstance(m, int):
        if m % 2:
            print(f'{m} is {int((m + 1) / 2)}th odd number')
        else:
            print(f'{m} is not an odd number')
```


3\. Слайсы массивов в *NumPy*

``` python
import numpy as np

a = np.arange(16).reshape(2, 2, 2, 2)

print(a[..., 0].flatten())
print(a[:, :, :, 0].flatten())

# output:
# [0  2  4  6  8 10 12 14]
# [0  2  4  6  8 10 12 14]
```
