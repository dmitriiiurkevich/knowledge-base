
### Определение

σ-Алгеброй, порождённой семейством измеримых отображений $\{f_i\}_{i \in I}$

$$f_i: (\mathcal X, \mathscr M) \rightarrow (\mathcal X_i, \mathscr M_i)$$

будем называть 

$$\Large \sigma \normalsize (f_i:i \in I) \stackrel{def}{=} \Large \sigma \normalsize \Big( \bigcup_{i \in I} \Large \sigma \normalsize (f_i) \Big)$$

То есть,
- наименьшую σ-алгебру, относительно которой все $f_i$​ измеримы
- наименьшую σ-алгебру, такую что все $\Large \sigma \normalsize (f_i)$ являются её подалгебрами

*N.B.*
Все отображения $f_i$ действуют из одного и того же пространства. Это важно.


*N.B.*
Ещё раз напомним. 

Объединение двух подалгебр может не быть подалгеброй.

$$\mathcal A \cup \mathcal B \ne \Large \sigma \normalsize(\mathcal A \cup \mathcal B)$$

Пересечение двух подалгебр не может не быть подалгеброй.

$$\mathcal A \cap \mathcal B = \Large \sigma \normalsize(\mathcal A \cap \mathcal B)$$



### Пример 2.5

$\mathcal X = \{1, 2, 3, 4, 5, 6\}$ — исходы броска игральной кости

$f: \mathcal X \to \{0,1,2\}$
$f(x)=x(mod \ 3)$

$g: \mathcal X \to \{0,1\}$
$g(x)=x(mod \ 2)$

$\Large \sigma \normalsize (f) = \{  \emptyset,\ \{1,4\},\ \{2,5\},\ \{3, 6\}, \ \{1,2,4,5\}, \ \{1,3,4,6\}, \ \{2,3,5,6\}, \ \mathcal X \}$

$\Large \sigma \normalsize (g) = \{  \emptyset,\ \{1,3,5\},\ \{2,4,6\}, \ \mathcal X \}$


$\Large \sigma \normalsize (f,g) = \Large \sigma \normalsize (\Large \sigma \normalsize (g) \ \cup \ \Large \sigma \normalsize (g)) = 2^{\mathcal X}$


Объединение двух неродственных подалгебр может породить полную алгебру.

Каждое из отображений $f$ и $g$ теряет информацию по-своему. Но если потери информации "ортогональны", то объединение может восстановить всё.

Китайская теорема об остатках — частный случай этого принципа: остатки по взаимно простым модулям вместе определяют остаток по составному модулю.


### Пример 1.5

$\mathcal X = \{1, 2, 3, 4, 5, 6\}$ — исходы броска игральной кости

$f: \mathcal X \to \{0,1,2,3\}$
$f(x)=x(mod \ 4)$


$g: \mathcal X \to \{0,1\}$
$g(x)=x(mod \ 2)$

$\Large \sigma \normalsize (f) = \{  \emptyset,\ \{1,5\},\ \{2,6\},\ \{3\}, \ \{4\}, \ \{1,2,5,6\}, \ \{1,3,5\}, \ \{1,4,5\}, \ \{2,3,6\}, \ \{2,4,6\}, \ \{2,4,6\}, \ \{1,2, 3, 5,6\}, \ \{1,2,4, 5, 6\}, \ \{2,3,4, 6\}, \ \mathcal X \}$
$\Large \sigma \normalsize (g) = \{  \emptyset,\ \{1,3,5\},\ \{2,4,6\}, \ \mathcal X \}$


$\Large \sigma \normalsize (f, g) = \Large \sigma \normalsize (f)$


В этом примере добавление $\Large \sigma \normalsize (g)$ не принесёт ничего нового, поскольку само отображение $g$ не несёт никакой дополнительной информации помимо той, что уже и так содержит в себе отображение $f$.


*N.B.*
$id: (\mathcal X, \mathscr M) \rightarrow (\mathcal X, \mathscr M)$
$id(x)=x$
$\Large \sigma \normalsize (id) = \mathscr M$

Тождественное отображение различает все точки, поэтому порождая исходную σ-алгебру.

$f: (\mathcal X, \mathscr M) \rightarrow (\mathcal X, \mathscr M)$
$f(x)=c$
$\Large \sigma \normalsize (f)=\{  \emptyset, \ \mathcal X \}$

Константное отображение склеивает все точки в одну, порождая тривиальную σ-алгебру.

Между этими крайностями — все остальные отображения, каждое со своей "разрешающей способностью".
