
Декоратор – это функция, принимающая на вход функцию и возвращающая другую функцию (или объект, допускающий функциональный вызов). Использование декораторов позволяет модифицировать поведение функций, не нарушая читаемости кода.

``` python
@trace                    
def identity(x):
	return x
```

``` python
def identity(x):              
	return x

identity = trace(identity)
```


### Как это работает?

В результате каждого декорирования создаётся новый объект замыкания функции-декоратора, ссылка на который помещается в переменную, в которой ранее хранилась ссылка на объект декорируемой функции.

Замыкание любой функции содержит в себе ссылки на все переменные из *enclosing* области видимости этой функции. В частности, в замыкании функции-декоратора будет содержаться переменная, в которой хранится ссылка на объект декорируемой функции — обычно эта переменная называется `func`.

Таким образом, если мы задекорируем одним и тем же декоратором несколько различных функций, у нас появятся несколько экземпляров замыкания функции-декоратора, каждое со своим собственным значением переменной `func`.

### Примеры полезных декораторов

##### time this
вычисляет среднее время выполнения функции

``` python
import time
import functools


def time_this(func=None, *, n_iter=100):
    if func is None:
        return lambda func: timethis(func, n_iter=n_iter)

    @functools.wraps(func)
    def inner(*args, **kwargs):
        acc = float("inf")
        for i in range(n_iter):
            tick = time.perf_counter()
            result = func(*args, **kwargs)
            acc = min(acc, time.perf_counter() - tick)
            print(acc)
        return result
    return inner
  
time_this(sum)(range(10**6))
```


##### profiled
вычисляет количество вызовов функции

``` python
import functools


def profiled(func):
   @functools.wraps(func)
   def inner(*args, **kwargs):
       inner.ncalls += 1
       return func(*args, **kwargs)
   inner.ncalls = 0
   return inner


@profiled
def identity(x):
   return x


identity(42)
identity(43)
identity(44)

print(identity.ncalls)
```


##### cached
кеширует результат первого вызова функции

``` python
import functools


def cached(func):
    called = False
    result = None

    @functools.wraps(func)
    def inner(*args, **kwargs):
        nonlocal called
        nonlocal result
        if not called:
            result = func(*args, **kwargs)
            called = True
        return result
    return inner


@cached
def identity(x):
   return x

print(identity(42))
print(identity(43))
```


##### memoized
сохраняет результаты выполнения функции для предотвращения избыточных вычислений
не будет работать, если хотя бы один из параметров функции не будет хешируемым

``` python
import functools


def memoized(func):
    cache ={}
    
    @functools.wraps(func)
    def inner(*args, **kwargs):
        key = args + tuple(sorted(kwargs.items()))
        if key not in cache:
            cache[key] = func(*args, **kwargs)
        return cache[key]
    return inner


@memoized
def identity(x):
   return x

print(identity(42))
```


##### singleton 1

``` python
import functools


def singleton(cls):
    instance = None

    @functools.wraps(cls)
    def inner(*args, **kwargs):
        nonlocal instance
        if instance is None:
            instance = cls(*args, **kwargs)
        return instance
    return inner
```

class C:

        if cls.instance is None:
