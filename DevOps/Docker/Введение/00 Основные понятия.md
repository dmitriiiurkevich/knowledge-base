
docker engine
- docker = docker CLI
- dockerd = docker daemon

docker ↔ REST API ↔ dockerd

Два варианта поставки:
- Docker Engine Community Edition
- Docker Engine Enterprise Edition — содержит встроенные функции безопасности и корпоративную поддержку. 


Registry — хранилище тегированных docker-образов, может быть 
- собственным 
- закрытым внутрикорпоративным 
- одним из сторонних:
	- Docker Hub
	- AWS Container Registry
	- Google Container Registкy
	- ...


Место registry в системе CI/CD:
- git commit →
- система непрерывной интеграции запускает новую сборку →
- если сборка прошла успешно, появляется новая версия контейнера →
- новая версия контейнера сохраняется в registry с новым тегом →
- registry отправляет уведомления о новой версии с новым тегом в другие системы →
- уведомление от registry запускает развёртывание в промежуточной среде →
- ...
