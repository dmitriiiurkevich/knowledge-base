Запускаем в *WSL 2* простейший *http*-сервер. 

```
$ python -m http.server
```

По умолчанию он стартует на 8000-м порту. Как пробросить порт из виртуальной машины *WSL* в *Windows*?

## Первый вариант
Вообще ничего делать не надо, начиная со второй версии *WSL*, *localhost* *Windows* будет видеть *localhost* виртуальной машины *WSL*.

## Второй вариант
1.
В линуксе, развёрнутом в *WSL*, выполняем команду
`$ ip a  | grep /20`

2.
 Получаем строку, начинающуюся примерно так:
 `inet 172.24.239.252/20`

Копируем *ip*-адрес.

3.
В *Windows* открываем под администратором файл *hosts* и создаём в нём запись
`172.31.94.239 wsl2`  

Заходим на сайт по адресу `wsl2:8000`

## Третий вариант
Пробрасываем порт с интерфейса Винды в WSL  
- Все одной строкой в cmd запущенном от администратора  
- 192.168.0.2 адрес в винде команда “ipconfig”  
- 172.31.94.239 адрес в WSL “ip a”  
netsh interface portproxy add v4tov4 listenaddress=192.168.0.2 listenport=8000 connectaddress=172.31.94.239 connectport=8000  
- Редактируемв винде в файле, под администраторм C:\Windows\System32\drivers\etc\host СВОЙ ip ПОЛУЧЕННЫЙ СТРОКОЙ ВЫШЕ, пример записи ниже  
- 172.31.94.239 homayk.ru  
- Заходим из винды на сайтег http://homayk.ru:8000  
- Из локальной сети заходим http://192.168.0.2:8000  
Для пытливых, и если позволяет роутер, можно настроить доменную запись там, для всех устройств в локальной сети.
