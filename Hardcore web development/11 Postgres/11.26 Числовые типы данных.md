
|          | байт | диапазон                                                |                 |
| -------- | ---- | ------------------------------------------------------- | --------------- |
| smallint | 2    | -32768 до 32767                                         | 32 тысячи       |
| int      | 4    | -2 147 483 648 до 2 147 483 647                         | 2 миллиарда     |
| bigint   | 8    | -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807 | $9\cdot10^{18}$ |

|                  | байт     |
| ---------------- | -------- |
| real             | 4        |
| double precision | 8        |
| numeric          | $\infty$ |

``` sql
select 1::numeric(5,2); --> 1
select 12::numeric(5,2); --> 12
select 123::numeric(5,2); --> 123
select 1234::numeric(5,2); --> ошибка — нужна точность 6
select 1234::numeric(6,2); --> 1 234
select 1234::numeric(6,0); --> 1 234
select 1234::numeric(6); --> 1 234
select 1234::numeric(5); --> 1 234
select 1234::numeric(4); --> 1 234

select 123.45::numeric(5,2); --> 123.45
select 1.2345::numeric(5,2); --> 1.23
select 1.111::numeric(5,2); --> 1.11
```

Применение математических функций может вызвать неявное приведение типов данных.

``` sql
select sin(21.53::numeric(5,2)); --> double precision  
select sin(21.53)::numeric(5,2); --> numeric
```

Конвертация строки в число

``` sql
select pg_typeof(2); --> int
select pg_typeof('2'::int); --> int
select pg_typeof('2'::bigint); --> bigint
select 1 + '2'::int;
```

Конвертация числа в строку

``` sql
select 156::text;
select pg_typeof(156::text); --> text
```
