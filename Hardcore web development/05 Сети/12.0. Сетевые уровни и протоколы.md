




Модель TCP/IP

| TCP/IP Layer      | Main protocol  | Header contains |              |          |
| ----------------- | -------------- | --------------- | ------------ | -------- |
| 0. Application    | HTTP           |                 |              |          |
| 1. Transport      | TCP, UDP       | TCP ports       | 2 bytes (×2) | 8 bytes  |
| 2. Internet       | IP             | IP addresses    | 4 bytes (×2) | 20 bytes |
| 3. Network Access | Ethernet, WiFi | MAC addresses   | 6 bytes (×2) |          |

Сетевое приложение можно реализовать так, чтобы оно общались только и непосредственно с Network Access уровнем стека TCP/IP. Это даёт максимальную свободу и позволяет экономить трафик, .
