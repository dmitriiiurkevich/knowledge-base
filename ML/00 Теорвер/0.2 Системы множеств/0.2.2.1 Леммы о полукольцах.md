
### Лемма 1

Семейства дизъюнктных подмножеств из третьего свойства полуколец — дизъюнктных подмножеств, дополняющих до внешнего множества — существуют не только для любого подмножества этого внешнего множества, но и для любого семейства подмножеств этого внешнего множества.

Пусть
$\mathscr S$ — полукольцо
$\mathcal A_1,...,\mathcal A_k \in \mathscr S$
$\mathcal A \in \mathscr S$

Тогда

$$\bigsqcup_{i=1}^k \mathcal A_i \subset \mathcal A$$
$$\Downarrow$$
$$\exists \ \mathcal B_{k+1},...,\mathcal B_n \in \mathscr S : \left( \bigsqcup_{i=1}^k \mathcal A_i \right) \sqcup \left( \bigsqcup_{j=k+1}^n \mathcal B_j \right) = \mathcal A$$


##### Доказательство

Индукция по $k$.

База: $k=1$ — следует из определения полукольца

Переход: $k \rightarrow k+1$
$\mathcal A$ 
$\mathcal A_1,...,\mathcal A_k \rightarrow \mathcal A_1,...,\mathcal A_k,\mathcal A_{k+1}$

По индукционному предположению для семейства подмножеств $\mathcal A_1,...,\mathcal A_k$ т. ч. $$\left( \bigsqcup_{i=1}^k \mathcal A_i \right) \subset \mathcal A$$
существует, расширяющее его семейство подмножеств $\mathcal B_{k+1},...,\mathcal B_n$ т. ч. $$\left( \bigsqcup_{i=1}^k \mathcal A_i \right) \sqcup \left( \bigsqcup_{j=k+1}^n \mathcal B_j \right) = \mathcal A$$
Все множества $\mathcal A_i$ и $\mathcal B_j$ попарно дизъюнктны
$\Rightarrow \mathcal A_{k+1} \subset \bigsqcup_{j=k+1}^n \mathcal B_j$

Введём обозначение: $\mathcal C_j = \mathcal A_{k+1} \cap \mathcal B_j, \ j=k+1,...,n$

$\mathcal C_j \in \mathscr S$
$\mathcal A_{k+1} = \bigsqcup_{j=k+1}^n \mathcal C_j$

$\mathcal C_j \subset \mathcal B_j \Rightarrow \exists \ \mathcal D_{j,1},...,\mathcal D_{j,l_j}: \mathcal B_j = \mathcal C_j \sqcup \left( \bigsqcup_{l=1}^{l_j} \mathcal D_{j,l} \right)$
$\Rightarrow \mathcal A = \left( \bigsqcup_{i=1}^k \mathcal A_i \right) \sqcup \left( \bigsqcup_{j=k+1}^n \mathcal C_j \right) \sqcup \left( \bigsqcup_{l=1}^{l_{k+1}} \mathcal D_{l}^{{k+1}} \right) \sqcup ... \sqcup \left( \bigsqcup_{l=1}^{l_n} \mathcal D_{l}^{n} \right)$
$\Rightarrow \mathcal A = \left( \bigsqcup_{i=1}^{k+1} \mathcal A_i \right) \sqcup \left( \bigsqcup_{l=1}^{l_{k+1}} \mathcal D_{l}^{k+1} \right) \sqcup ... \sqcup \left( \bigsqcup_{l=1}^{l_n} \mathcal D_{l}^{n} \right)$

### Лемма 2

Любое конечное семейство элементов полукольца, как бы они ни пересекались друг с другом, всегда можно "нарезать" на дизъюнктные части. 

$\mathscr S$ — полукольцо
$\mathcal A_1,...,\mathcal A_n \in \mathscr S$ — произвольный конечное семейство элементов полукольца
$\Rightarrow \exists \ \mathcal B_{1},...,\mathcal B_m \in \mathscr S :$
- $\forall i,j \in \{1..n\} \ \mathcal B_i \cap \mathcal B_i = \emptyset$ — все $\mathcal B_i$ дизъюнктны
- $\forall i \in \{1 ... n\} \ \exists \ \Gamma_i \subset \{1..m\}: \mathcal A_i =  \bigsqcup_{j \in \Gamma_i} \mathcal B_j$

##### Доказательство

Индукция по $n$.

База: $n=1$ — у нас есть только одно множество $\mathcal A_1$, и оно же, в таком случае, и будет единственным элементом искомого семейства дизъюнктных множеств $\mathcal B_i$
$$m=1, \ \mathcal B_1 = \mathcal A_1$$

Переход: $n \rightarrow n+1$
$\mathcal A_1,...,\mathcal A_{n} \rightarrow \mathcal A_1,...,\mathcal A_{n},\mathcal A_{n+1}$

По индукционному предположению для семейства произвольных подмножеств $\mathcal A_1,...,\mathcal A_{n}$
существует семейство дизъюнктных подмножеств $\mathcal B_{1},...,\mathcal B_m \in \mathscr S$ т. ч.

$$\mathcal A_1,...,\mathcal A_{n} \leadsto \{\mathcal B_{j} \}_{j=1}^{m}$$
$$\mathcal A_i = \left( \bigsqcup_{j \in \Gamma_i} \mathcal B_j \right)$$

Добавляем новое подмножество $\mathcal A_{n+1}$ 

Пересекаем подмножество $\mathcal A_{n+1}$ со всеми $\mathcal B_i$
Получаем семейство дизъюнктных подмножеств $\mathcal C_i = \mathcal A_{n+1} \cap \mathcal B_i \in \mathscr S , \ i \in \{1 ... m \}$


С одной стороны, $\mathcal C_i \subset \mathcal A_{n+1}$

По предыдущей лемме существует семейство дизъюнктных элементов полукольца $\mathcal D_1,...,\mathcal D_l \in \mathscr S$ дополняющий семейство дизъюнктных подмножеств $\mathcal C_1,...,\mathcal C_m$ до $\mathcal A_{n+1}$ 

То есть,
$$\mathcal A_{n+1} = \left( \bigsqcup_{i=1}^{m} \mathcal C_i \right) \sqcup \left( \bigsqcup_{k=1}^{l} \mathcal D_k \right) $$


С одной стороны, $\mathcal C_i \subset \mathcal B_i$

По определению полукольца для каждого $i \in \{ 1,...,m\}$ существует семейство дизъюнктных элементов полукольца $\mathcal B_{j,1},...,\mathcal B_{j,P_j}$ дополняющий $\mathcal C_i$ до $\mathcal B_i$

$$\mathcal B_i = \mathcal C_i \sqcup \left( \bigsqcup_{j=1}^{m} \bigsqcup_{p=1}^{P_j} \mathcal B_{j,p} \right) $$

То есть,
$$\mathcal A_1,...,\mathcal A_{n+1} \leadsto \{ \mathcal C_i \}_{i=1}^{m} \ \cup \ \{ \mathcal D_k \}_{k=1}^{l}  \ \cup \ \{\{ \mathcal B_{j,p} \}_{p=1}^{Pj}\}_{j=1}^{m} $$


Замечание
Не умаляя общности, можно считать, что семейство $\mathcal B_{1},...,\mathcal B_m$ не содержит "лишних" подмножеств, не участвующих в разложении какого-либо из подмножеств $\mathcal A_1,...,\mathcal A_{n}$
