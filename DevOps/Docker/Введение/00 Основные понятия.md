
docker engine
- docker = docker CLI
- dockerd = docker daemon

docker ↔ REST API ↔ dockerd

Два варианта поставки:
- Docker Engine Community Edition
- Docker Engine Enterprise Edition

Последний содержит встроенные функции безопасности и предполагает соблюдение соглашения об уровне обслуживания корпоративного класса. 

Service Level Agreements — SLA — это соглашение между клиентом и поставщиком об уровне предоставляемой  услуги, который выражается в измеримых показателях. Нарушение SLA влечет за собой штрафные санкции. 


Registry — хранилище тегированных docker-образов, может быть собственным закрытым внутрикорпоративным, или одним из сторонних:
- Docker Hub
- AWS Container Registry
- Google Container Registкy
- ...

Место registry в системе CI/CD:
- git commit
- система непрерывной интеграции запускает новую сборку
- если сборка прошла успешно, появляется новая версия контейнера
- новая версия контейнера сохраняется в registry с новым тегом 
- registry отправляет уведомления о новой версии в другие системы
- уведомление от registry запускает развёртывание в промежуточной среде
- ...
