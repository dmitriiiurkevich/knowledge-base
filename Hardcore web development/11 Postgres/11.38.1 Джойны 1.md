
Существуют 5 основных видов *джойнов*:
- *INNER JOIN* = *JOIN*
- *LEFT OUTER JOIN* = *LEFT JOIN*
- *RIGHT OUTER JOIN* = *RIGHT JOIN*
- *FULL OUTER JOIN* = *FULL JOIN* 
- *CROSS JOIN*

И 5 дополнительных возможностей *SQL*, которые не являются отдельными видами *джойнов*, но относятся к этой теме:
- *SELF JOIN*
- *NATURAL JOIN*
- *JOIN LATERAL*
- *SEMI JOIN*
- *ANTI JOIN*

Замечание 0
Слова *INNER* и *OUTER* не нужны и, как правило, не используются.

*CROSS JOIN* — это декартово произведение двух таблиц.

``` sql
select * from t1, t2;
select * from t1 cross join t2;
select * from t1 join t2 on true;
```


*LEFT JOIN* можно определить так:

``` sql
select t1.id, t2.id 
from t1    
cross join t2   
where t1.id = t2.id

union all

select t1.id, null   
from t1   
where not exists (
	select
	from t2
	where t2.id = t1.id   
)
```

*RIGHT JOIN* можно определить так:

``` sql
select t1.id, t2.id 
from t1    
cross join t2   
where t1.id = t2.id

union all

select null, t2.id   
from t2   
where not exists (
	select
	from t1
	where t1.id = t2.id   
)
```


Замечание 1
*RIGHT JOIN* не нужен и, как правило, очень редко используется. Всегда можно изменить порядок объединяемых таблиц таким образом, чтобы использовать только *LEFT JOIN*.

Аналогично определяются *INNER JOIN*

``` sql
select t1.id, t2.id 
from t1    
cross join t2   
where t1.id = t2.id
```

и *FULL JOIN*
