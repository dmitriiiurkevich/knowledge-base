*VARCHAR*
*TEXT*

Производительность у типов данных *VARCHAR* и *TEXT* одинаковая, *VARCHAR* может иметь опциональное ограничение по длине строки.

CHAR

Строка фиксированной длины, с автоматически  дополняющаяся пробелами.

Length

``` sql
select length('hello, world'); --> 12
```


``` sql
select book_id, name from book order by length(name) desc;
```

Concatenation

``` sql
select 'Book: ' || name as name from book;
```

Formatting

``` sql
select format('Book: %s!', 'The Lord of the Rings');
select format('Author: %s!', name) from author;


select format('%s by %s', b.name, a.name) as book_by_author
from book b
join author a using(author_id)
order by a.name, b.name;
```


Substrings

``` sql
select substring('hello, world' from 1 for 5); --> hello
```

*N.B.*: нумерация начинается с единицы, правая граница включается — всё не как у людей.

``` sql
select replace('hello, world', 'hello', 'bye');
```

``` sql
select position('hello' in 'hello, world'); --> 1

select position('Hello' in 'hello, world'); --> 0
```

Register

``` sql
select upper(name) from author;
select lower(name) from author;
```

Casting

``` sql
select '123'::int; --> ошибка

select cast('123' as int); --> ошибка
```

Splitting

``` sql
select split_part('2024-06-24', '-', 1); --> 2024
select split_part('2024-06-24', '-', 2); --> 06
select split_part('2024-06-24', '-', 3); --> 24

select split_part('2024-06-24', '-', -3); --> 2024
select split_part('--2024-06-24', '-', -4); --> пустая строка
```

По результату селекта мы не можем понять: идут ли два дефиса подряд, или мы вышли за пределы диапазона.

``` sql
select split_part('----', '-', -4); --> пустая строка
select split_part('----', '-', -5); --> пустая строка
select split_part('----', '-', -6); --> пустая строка
```
