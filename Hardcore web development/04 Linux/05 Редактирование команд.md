Ctrl + a, Home — в начало команды
Ctrl + e, End — в конец команды
Fn + ↔ — то же самое на маке

Ctrl + w — удалить литерал от текущей позиции до первого пробела слева
Ctrl + y — вставить ранее удалённый литерал на текущую позицию

Ctrl + ↔, Alt + ↔ — влево/вправо на одно слово
Option + ↔ — то же самое на маке

При желании эти сочетания клавиш могут быть переопределены

```shell 
~$ cat ~/.inputrc
# output:
# "\C-b":backward-word
# "\C-f":forward-word
```
