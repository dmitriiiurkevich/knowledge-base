
`$ ls -1` — возвращает список файлов одной колонкой
`$ ls -a` — включая скрытые файлы
`$ ls -l` — *long format*
`$ ls -h` — *human readable* — человеко-читаемый размер файлов 

`../` — директория на 1 уровень выше
`../../` — директория на 2 уровня выше
`../../../` — директория на 3 уровня выше
...

`$ find my_current_path -type d` — возвращает рекурсивный список директорий, вглубь, начиная с *my_current_path* 

`$ find my_current_path -type d -ls | less` — возвращает рекурсивный список директорий, вглубь, начиная с *my_current_path*, снабжённый дополнительными параметрами

`$ tree my_current_path` — возвращает дерево директорий, вглубь, начиная с *my_current_path* 

## Создание файлов и директорий

`$ touch my_file` — создаст файл
`$ touch my_file_1 my_file_2` — создаст несколько файлов

`$ mkdir my_dir` — создаст директорию
`$ mkdir -p my_dir/my_subdir/my_subdir` — создаст цепочку вложенных директорий

## Поиск файлов и директорий

`$ find my_dir -type f -name "my_file"`
`$ find my_dir -type d -name "my_subdir"` — поиск по имени


`$ grep -rnw 'my_dir_path' -e 'content_pattern'` — поиск файла по содержимому
	`-r` — search recursively
	`-n` — add line number to the response
	`-w` — stands for match the whole word


## Навигация по директориям

`$ cd my_directory` — перейдёт в директорию *my_directory*
`$ cd ~` или `$ cd` — перейдёт в домашнюю директорию пользователя
`$ cd -` — перейдёт в предыдущую директорию

## Размер директорий

`$ du -sh my_directory` — вернёт размер директории
`$ du -sh my_directory/*` — вернёт размер всех нескрытых файлов директории (вместо * можно использовать маску)
`$ du -sh my_directory/* | sort -h | less` — вернёт размер всех нескрытых файлов в порядке убывания их размера через пейджер *less*
